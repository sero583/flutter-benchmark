Native Platform Benchmarks

The same app runs natively on Android, iOS, Windows, Linux, and macOS. Use --dart-define=RENDERER=name to label the renderer.

# Android  (Skia default, Impeller opt-in)
flutter run -d android
flutter run -d android --enable-impeller --dart-define=RENDERER=Impeller

# iOS  (Impeller default since Flutter 3.16)
flutter run -d ios
flutter run -d ios --no-enable-impeller --dart-define=RENDERER=Skia

# Windows
flutter run -d windows
flutter run -d windows --enable-impeller --dart-define=RENDERER=Impeller

# Linux
flutter run -d linux

# macOS  (Impeller default)
flutter run -d macos
flutter run -d macos --no-enable-impeller --dart-define=RENDERER=Skia
